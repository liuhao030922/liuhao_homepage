# CV Upload Instructions

1. Put your PDF CV file in this folder:
   - `assets/files/`
2. Name the file:
   - `Hao_Liu_CV.pdf`
3. The website is already configured to link this path:
   - `/assets/files/Hao_Liu_CV.pdf`

If you want to use another filename, update this field in:
- `_data/profile.yml`

```yaml
cv_link: /assets/files/Your_File_Name.pdf
```
